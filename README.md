# Simple Digital Clock with Vanilla JS

A minimalist, clean, and responsive digital clock web application built using HTML, CSS, and vanilla JavaScript.

---

## 📌 Project Overview

This is a small-scale, hobby project created to practice core web development concepts. The main focus of this project was to master asynchronous JavaScript timing events and DOM manipulation by creating a real-time updating clock and day display.

(Live Demo)[]

## 💡 What I Learned

By building this project, I successfully learned and implemented:

- **`setInterval()` Method:** Understanding how to execute a function repeatedly with a fixed time delay (every 1000 milliseconds) to keep the clock ticking in real-time.
- **The `Date` Object:** Extracting real-time hours, minutes, seconds, and days of the week using JavaScript's built-in `Date` methods.
- **Ternary Operators & String Padding:** Formatting time strings (e.g., converting single-digit numbers like `9` into `09` for a standard digital clock format).
- **DOM Manipulation:** Dynamically updating the HTML structure using `innerHTML` and `textContent`.

---

## 🛠️ Tech Stack

- **HTML5** - Markup structure
- **CSS3** - Flexbox styling and typography layout
- **JavaScript (ES6+)** - Time logic and dynamic rendering
